# 🧠 智能英语学习助手

一个基于前沿记忆科学的智能英语学习HTML应用，结合间隔重复算法、多模态学习和游戏化设计，帮助您高效记忆英语单词。

## ✨ 核心特性

### 🎯 科学记忆算法
- **间隔重复算法**: 基于艾宾浩斯遗忘曲线，智能计算最佳复习时间
- **自适应学习**: 根据您的掌握程度动态调整学习难度
- **记忆强化**: 多种测试模式强化记忆印象

### 📚 多样化学习模式
- **卡片学习**: 翻转卡片显示英文→中文→音标→例句
- **智能测试**: 英译中/中译英选择题、填空题、听音辨词
- **语音合成**: 标准英式/美式发音，支持语音学习
- **例句学习**: 在语境中理解和记忆单词

### 🎮 游戏化激励系统
- **积分系统**: 学习获得积分，连续学习获得奖励倍数
- **等级系统**: 根据积分升级，解锁新功能
- **成就徽章**: 连续学习天数、学习目标等多种成就
- **学习统计**: 可视化进度追踪和学习分析

### 💻 现代化体验
- **响应式设计**: 完美支持手机、平板、电脑
- **暗色主题**: 护眼的暗色/亮色主题切换
- **键盘快捷键**: 空格翻卡片、方向键导航等
- **离线可用**: 纯前端实现，无需网络连接

## 🚀 快速开始

### 方法一：直接使用
1. 下载 `english-learning-app.html` 和 `advanced-features.js` 两个文件
2. 将两个文件放在同一个文件夹中
3. 双击打开 `english-learning-app.html` 即可开始使用

### 方法二：导入您的词汇
1. 准备您的JSON词汇文件（格式见下方）
2. 点击"导入词汇文件"按钮选择文件
3. 开始个性化学习

## 📄 词汇文件格式

```json
[
  {
    "id": 1,
    "english": "example",
    "chinese": "例子",
    "phonetic": "/ɪɡˈzæmpl/",
    "forms": {
      "plural": "examples (复数)"
    },
    "example": {
      "en": "This is an example sentence.",
      "cn": "这是一个例句。"
    }
  }
]
```

### 字段说明：
- `id`: 唯一标识符（数字）
- `english`: 英文单词（必需）
- `chinese`: 中文翻译（必需）
- `phonetic`: 音标（可选）
- `forms`: 词形变化（可选）
- `example`: 例句对象，包含英文和中文例句

## 🎯 使用指南

### 卡片学习模式
1. **查看单词**: 显示英文单词和音标
2. **点击翻转**: 查看中文翻译和例句
3. **语音播放**: 点击🔊按钮或按S键播放发音
4. **标记难度**: 根据掌握程度选择"简单"、"一般"或"难记"
5. **智能复习**: 系统会根据您的标记安排最佳复习时间

### 智能测试模式
- **选择题**: 英译中/中译英选择题
- **填空题**: 根据例句完成填空
- **听音辨词**: 听发音选择正确单词
- **即时反馈**: 答题后立即显示结果和积分

### 学习统计
- **进度追踪**: 查看已学词汇数量和掌握情况
- **学习日历**: 可视化显示每日学习情况
- **准确率分析**: 测试准确率和平均反应时间
- **学习建议**: 基于数据的个性化学习建议

### 成就系统
- **初学者**: 学习第一个单词
- **坚持不懈**: 连续学习3天
- **一周达人**: 连续学习7天
- **积分大师**: 获得500积分
- **完美答题**: 测试连续答对10题

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 翻转卡片 |
| `←` | 上一个单词 |
| `→` | 下一个单词 |

| `1` | 标记为"难记" |
| `2` | 标记为"一般" |
| `3` | 标记为"简单" |

## 🔧 技术特点

- **纯前端实现**: HTML5 + CSS3 + JavaScript，无需服务器

### 自动保存
